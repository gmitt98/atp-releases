# ATP — Adaptive Training Platform

A local-first training app for cyclists and runners. **Your data stays on your machine** — nothing
is uploaded anywhere.

## Download

Grab the latest build for your computer from the [**Releases**](../../releases/latest) page:

| Your computer | File |
|---|---|
| **Mac** (Apple Silicon — M1/M2/M3/M4) | `ATP-macos-arm64.zip` |
| **Windows** | `ATP-windows-x64.zip` |

Unzip and open. The app isn't code-signed, so the **first** launch needs a one-time "allow" step
(right-click → Open on Mac; More info → Run anyway on Windows) — the steps are on each release page.

Once installed, the app checks here for new versions and shows an in-app banner when one is out, so
you only download from here the first time.

---

*This repository contains only the built apps and the `latest.json` update manifest — no source code.*
